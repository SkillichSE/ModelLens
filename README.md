# ⚔️ ModelArena

**[skillichse.github.io/ModelArena](https://skillichse.github.io/ModelArena)**

Free AI models, ranked every day. No paywalls, no hype — just numbers.

---

ModelArena runs automatically every night and publishes fresh results. It answers one question: **which free AI model is actually the best right now?**

Every model is tested on things that matter in practice — writing correct code, solving logic problems, following instructions, translating text. Code is actually executed and checked against expected output. Reasoning questions have known correct answers. Nothing is graded by vibes.

Speed and quality are always shown separately. A small fast model and a large smart model can both be number one — in their own category and size tier.

All models tested are completely free to use. No credit card required on any provider.
