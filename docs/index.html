Why ModelArena?

Other benchmarks compare paid models. We focus exclusively on what you can use for free right now.

Real code execution Code is actually run against test cases — not just checked for syntax. A model that writes broken code scores 0.
Verified answers Reasoning questions have known correct answers. No subjective grading — either the model got it right or it didn't.
Fair size categories 4B models compete against other small models, not 120B giants. Find the best in your weight class.
Updated daily Runs every night via GitHub Actions. If a model gets deprecated or a new one launches, rankings update within 24h.
